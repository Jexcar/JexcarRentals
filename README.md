# Solar-Powered Rental Trailer Fleet

This is the project structure for the solar-powered rental trailer fleet website.

## Folder Structure

```
/solar-powered-rental-trailer-fleet
|-- frontend
|   |-- (Frontend code and assets)
|-- backend
|   |-- (Backend code and assets)
|-- documentation
|   |-- (Documentation files)
|-- config
|   |-- (Configuration files)
```

## Description

- **frontend**: This directory will contain all frontend-related code and assets.
- **backend**: This directory will hold backend code and configurations.
- **documentation**: All relevant documentation files will be stored here.
- **config**: This folder will include configuration files used in the project.
